¿Qué es CSS?

CSS son las siglas en inglés de Cascading Style Sheets, que significa «hojas de esilo en cascada».

Es un lenguaje utilizado para estandarizar elementos escritos en un lenguaje de marcado como HTML.

CSS fue creado por el W3C (World Wide Web Consortium) en 1996 por una razón. HTML no fue diseñado

para proporcionar etiquetas que ayuden en el formato de la página. Esto solo se hace para escribir

la marca registrada del sitio.

La inclusión de etiquetas, como en la versión 3.2 de HTML, causó varios problemas a los desarrolladores.

Debido a que los sitios web tienen fuentes, colores de fondo y estilos diferentes, el proceso de reescribir

el código fue largo, doloroso y costoso. Como resultado, el W3C desarrolló CSS ​​para abordar este problema.
